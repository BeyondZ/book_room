#### 使用技术

* 后端：Spring + Spingmvc + Mybatis + Redis + 阿里云短信

* 前端：jsp + easyui + jquery + ajax + echarts

* 运行环境：阿里云 + Docker

#### 部署
* 运行 project_hotel.sql 
* 将 web / src / main / resource / db.properties 的数据库账号密码改对
* 在任何一个servlet服务器运行web模块
* 访问index.jsp即可
